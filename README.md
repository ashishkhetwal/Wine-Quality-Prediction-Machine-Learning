# Wine Quality Prediction System

A Machine Learning project that predicts wine quality based on its chemical properties using classification models.

This project helped me understand how structured tabular data can be used for classification using tree-based algorithms.

## Concepts Used

* Data Preprocessing  
* Label Binarization  
* Correlation Analysis  
* Heatmap Visualization  
* Decision Tree Classifier  
* Random Forest Classifier  
* Train-Test Split  
* Binary Classification  

## Results

* Built Decision Tree and Random Forest models for wine quality prediction.  
* Converted wine quality into binary classes using label binarization.  
* Used correlation analysis and heatmap to understand feature relationships.  
* Achieved **92.2% accuracy** on the test dataset.  

## What I Learned

Through this project, I understood how Decision Tree and Random Forest classifiers work for classification problems and how preprocessing techniques like label binarization and correlation analysis help in analyzing data patterns.
